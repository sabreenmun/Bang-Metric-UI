Bang Metric Calculator

ABOUT

This calculator is based on Demarco's Bang Function. It calculates the "Bang Metric" by evaluating function primitives and their associated token counts. The token counts have preset weights, and each function primitive is assigned a weight based on its class. These weights are predefined for different classes of primitives and are used in the calculation of the Bang Metric score.

Weights for function primitives:

  • Separation: 0.6
  • Amalgamation: 0.6
  • Switching: 0.3
  • Simple Update: 0.5
  • Storage Management: 1.0
  • Edit: 0.8
  • Coherency: 1.0
  • Text Manipulation: 1.0
  • Synchronization: 1.5

  • Display: 1.8
  • Summarizing: 1.0
  • Arithmetic: 0.7
  • Initiation: 1.0
  • Computation: 2.0
  • Device Management: 2.5

HELP GUIDE

  • Click 'Add Primitive' to add a new function primitive.
  • Enter the token count for each function primitive you add.
  • You can add as many primitives as needed.
  • To remove a primitive, click the 'x' button next to it.
  • Once all fields are filled out, click 'Calculate' to get the Bang Metric score and classification.
  • Click 'Clear' to reset the form.
  • If you wish to save the data, click 'Save'.
